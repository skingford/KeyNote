# KeyNote - 技能树知识管理平台

## 产品概述

KeyNote 是一款面向技术人员的技能树知识管理和职业发展平台。帮助用户系统化管理多技术栈知识体系、跟踪学习进度、智能复习、AI 辅助面试准备，并实时生成简历。

**目标用户**：技术开发者、求职者、终身学习者

**平台**：Web 应用（响应式，支持桌面 + 平板 + 手机）

**技术栈**：Next16.1.6 + React 19.2.4 + TypeScript + shadcn ui

## 交互规范

### 技能树交互
- **桌面端**：鼠标滚轮缩放，拖拽平移，悬停显示详情，双击进入编辑
- **移动端**：双指缩放，单指拖拽，点击显示底部详情面板，长按弹出操作菜单

### 闪卡复习交互
- **桌面端**：点击翻转，键盘 1-5 评分，← → 切换
- **移动端**：点击翻转，左右滑动切换，底部按钮评分

### 表单交互
- **桌面端**：弹窗/侧边抽屉
- **移动端**：全屏页面 / 底部弹出面板

### 加载状态
- 骨架屏（Skeleton）用于页面加载
- 旋转加载器用于操作等待
- 进度条用于文件上传/导出
